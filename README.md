# wordle_solver

A Wordle Clone made using Flutter

## Features
- [x] Unlimited Words
- [x] Lifetime Stats
- [x] Same words for all users (Daily)
- [ ] Remembers which word puzzle you're on
- [ ] Game Instructions
- [ ] Optimal Wordle Solver using Infomation Theory (Inspired by 3Blue1Brown)
- [ ] Share to Socials
